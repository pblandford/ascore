Note Input

The basics of note input are simple - just press the notes on the keyboard and watch them appear on the score:

The notes appear at the current position of the note input marker, which moves along as they are entered. To reposition this marker, tap once at the required location.

If you do not wish the marker to move, long-press on the note - in this way chords can be easily entered.

The marker can be moved forwards and backwards with the arrow buttons:

If you make a mistake, use the 'undo' button to go back the previous state:

To redo any action, just long-press the same button.

You can select different durations by moving the left and right arrows of the duration selector:

Dotted durations can be selected with the dot selector - long-press for double-dotted notes:

To choose an accidental (eg whether a note will be F# or Gb), use the accidental selector:

To enter an accidental on white notes, eg a B# or Fb, use the underlined accidentals:

A rest can be entered with the rest button - the duration of the rest is determined by the duration selector.

To delete notes, position the marker at the note you wish to delete, and press the delete button:
