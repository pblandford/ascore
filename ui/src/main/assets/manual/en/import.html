Importing Files

Various types of files can be imported:

  • A-Score Save Files (with an *.asc suffix)
  • Music XML Files (*.xml or *.mxl)
  • Soundfont Files (currently *.sf2 only)
  • Text font Files (*.ttf and *.otf)

There are two ways to import:

  • From the menu, go to Export/Import -> Import, and find the file from the file picker
  • Open a file browser app, Google Drive, Dropbox or similar, and find the file - depending on the app, either just click on the file or select 'Open With', and A-Score should appear in the list of apps that can open the file.

A-Score and Music XML files will open up a new score if they are valid. Soundfont and text files will be stored away and available to use when required. When you create a new score or edit an instrument, you should see the imported soundfont instruments in the list of available sounds. Similarly, when editing a piece of text, the import text font file should be there to choose.
