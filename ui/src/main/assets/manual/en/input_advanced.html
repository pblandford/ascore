Note Input - Advanced

Pressing the up arrow button reveals a series of options relating to aspects of note input:

The controls on the left allow you to input grace notes - the leftmost for appogiaturas, the right one for acciaccaturas (ie grace notes with a slash through them):

The arrow button determines what happens when the marker is place on a grace note and a new grace note is entered - when selected, it will shift later grace notes to make room for the new note.

When unselected, it will add the new note to the one at the marker to form a chord.

Selecting the octave button adds an extra note an octave below each note entered - useful for piano music:

Selecting the dotted rhythm button inputs notes in dotted rhyhtm - saving the tedium of constantly having to toggle the dot and duration buttons:

Selecting the tie button creates a tie between the next note entered, and the previous one - provided the previous chord has a note in common with it:

The notehead button allows the style of note head to be altered (normal, cross or diamond):

The note size button allows the input of small note heads (eg for cues):

The no-input button allows notes to be sounded, without being entered - useful for trying out ideas without committing them to the score.
