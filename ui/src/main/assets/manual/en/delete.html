Deleting events

There are several ways to remove events.

Method 1

In input mode (ie, when the keyboard is showing), position the marker at the required segment and press the eraser button

This will delete any chord at the marker and move the marker along one. If there is a rest at the marker, it will be merged with any neighbouring rest.

Method 2

Long press on the eraser button - this will put the app into a special delete mode, where any event you touch in the score will be removed.

Method 3 - deleting specific event types

Navigate to the insert page for the event you want to delete, then press the eraser button - this will put the app into a special mode similar to the last method, except only events of the desired type will be deleted. This is useful when the events are very small, for example staccato marks.

Deleting a range of events

Long press in score to select the start of the range, then tap once to select the end. Then press the eraser button - this will remove all notes, and consolidate rests as far as possible. It will also remove other events such as lyrics, harmonies and dynamic marks - some events, such as time signatures and key signatures, are not affected.

Deleting Bars

Long press in the score to select the first bar you want to delete, then tap to select the end of the range. Then long-press the delete button to remove all bars in the range.

Deleting Instruments

Navigate to the Insert Instrument page and press the delete button - this will turn gray, putting the app into its 'delete event' mode. Now any instrument you press on will be removed.
